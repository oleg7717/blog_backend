# Приложение-блог
Серверная часть приложения-блога, разработанная с использованием Spring Framework
Для запуска необходимо выполнить:
1. Команду сборки war файла приложения

   $ gradle clean war
2. Заменить <user_placeholder> и <password_placeholder> на логин и пароль от базы данных в файле docker-compose.yaml в корне проекта
3. Выполнить команду сборки и запуска docker контейнеров 

   $ docker compose --build up
 
Приложение будет доступно по адресу:

      http://localhost:8080

Пример использования для получения поста с идентификатором 2:

      http://localhost:8080/api/posts/2
